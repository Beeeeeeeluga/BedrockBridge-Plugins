# BedrockBridge Plugins Collection 🎮

Welcome to my custom BedrockBridge plugins repository!
This project contains a collection of custom-built bridge plugins designed to enhance your Minecraft Bedrock Dedicated Server (BDS) and streamline your Discord integration.

> **💡 About BedrockBridge**
> All plugins in this repository are built on top of the **BedrockBridge** framework developed by [InnateAlpaca](https://github.com/InnateAlpaca/BedrockBridge).
> Please make sure you have BedrockBridge successfully installed and configured on your server before using these plugins.

---

## ⚡ Plugin Versions

To support different server environments and preferences, we provide two main versions of our plugins:

### 🚀 v2.0.0 (Optimized Version) - *Recommended*
* **Development & Optimization:** Enhanced with performance improvements and structural refactoring.
* **Key Improvements:**
  * Significantly reduced latency for background API calls.
  * Resolved potential Discord message synchronization delays during high player concurrency.
  * Optimized memory footprint to keep your BDS running light and stable.
* **Target Audience:** Servers looking for maximum performance, high stability, and compatibility with the latest BedrockBridge updates.

### 📜 v1.0.0 (Legacy Version) - *Archive*
* **Development:** The original custom plugin release.
* **Features:** Pure and stable implementation of the core features.
* **Target Audience:** Developers or server owners who need compatibility with older environments or want to inspect the original code logic.
