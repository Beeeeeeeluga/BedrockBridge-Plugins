🚀 INSTALLATION - 3 SCHRITTE
═════════════════════════════════════════════════════════════════════════════

SCHRITT 1: Webhook-URLs besorgen (5 Min)
└─ Lese: START_HERE.md
└─ Öffne Discord → Channel → Webhooks → New Webhook → URL kopieren

SCHRITT 2: main.js konfigurieren (2 Min)
└─ Öffne: main.js
└─ Zeile ~25: Webhook-URLs eintragen
└─ Zeile ~135: Server-Name eintragen
└─ Zeile ~45: Features einschalten/ausschalten

SCHRITT 3: Plugin laden (1 Min)
└─ Öffne: D:\BB\bridgePlugins\index.js
└─ Füge ein: import "./webhookbridge/main.js"
└─ Server neu starten

FERTIG! 🎉
└─ Test: !webhook test
└─ Prüfe Discord auf Test-Nachricht

🎯 SCHRITT-FÜR-SCHRITT: VON ALT ZU NEU
═════════════════════════════════════════════════════════════════════════════

Wenn du die alte Struktur hast und zu neu wechseln willst:

1. BACKUP MACHEN
   tar -czf webhookbridge-old.tar.gz core/ api/ events/ *.js

2. ALTE DATEIEN LÖSCHEN
   rm -rf core/ api/ events/
   rm config.js config-enhanced.js discord-webhook-enhanced.js utils-enhanced.js

3. index.js.new KOPIEREN
   cp index.js.new index.js

4. main.js KONFIGURIEREN
   Öffne main.js und trage Webhook-URLs ein

5. SERVER NEU STARTEN

6. TESTEN
   !webhook test
   Prüfe Discord auf Test-Nachricht
