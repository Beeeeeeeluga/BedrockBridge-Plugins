🚀 INSTALLATION (5 MINUTEN)
═════════════════════════════════════════════════════════════════════════════

SCHRITT 1: Discord Webhooks erstellen
   → Öffne deinen Discord Server
   → Rechtsklick auf Channel → Edit Channel → Webhooks
   → Create Webhook → Kopiere die URL
   → Wiederhole für verschiedene Channels (Logs, Errors, etc.)

SCHRITT 2: Dateien installieren
   → Kopiere diese 3 Dateien ins webhookbridge Verzeichnis:
     • discord-webhook-enhanced.js
     • config-enhanced.js
     • utils-enhanced.js

SCHRITT 3: Konfigurieren
   → Öffne config-enhanced.js
   → Ersetze Webhook-URLs
   → Stelle Server-Name ein
   → Wähle Features

SCHRITT 4: Plugin laden
   → Öffne index.js
   → Füge ein: import "./webhookbridge/discord-webhook-enhanced.js"

SCHRITT 5: Testen
   → Server neu starten
   → Im Chat: !webhook test
   → Prüfe Discord auf Test-Nachricht ✅

⚙️ SCHNELLE KONFIGURATIONEN
═════════════════════════════════════════════════════════════════════════════

Nur Server-Status (Minimal):
   features: {
     server: { startStop: true },
     analytics: { enabled: true }
   }

Standard Public Server:
   features: {
     chat: { enabled: true },
     players: { joinLeave: true },
     combat: { deathMessages: true }
   }

Vollständiges Monitoring:
   (Alle Features: true)

Performance-Optimiert:
   advanced.performance.messageBatchSize = 20
   advanced.performance.messageQueueDelay = 2000

🆘 HÄUFIGE PROBLEME
═════════════════════════════════════════════════════════════════════════════

❌ Webhooks funktionieren nicht
   ✅ !webhook test ausführen
   ✅ Discord Berechtigungen prüfen
   ✅ Webhook URL validieren
   ✅ Siehe: README-ENHANCED.md Troubleshooting

❌ Messages kommen nicht an
   ✅ Test-Mode deaktivieren (testMode: false)
   ✅ !webhook status prüfen
   ✅ Errors Webhook konfigurieren
   ✅ Debug-Mode: debug.enabled = true

❌ Performance-Probleme
   ✅ Weniger Features aktivieren
   ✅ Batch-Size erhöhen
   ✅ Message-Delay anpassen
   ✅ Cache-TTL überprüfen
