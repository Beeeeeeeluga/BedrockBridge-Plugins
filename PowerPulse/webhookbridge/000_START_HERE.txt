API METHODS (NEW)
================================================================================

From any plugin, you can now use:

  getPlayerStats(playerName)     → Get player statistics
  getServerReport(type)          → Get daily/weekly/monthly reports
  queryEvents(criteria)          → Query archived events
  getTopPlayers(metric, limit)   → Get top players ranking
  exportStatistics()             → Export all statistics
  getServerUptime()              → Get server uptime
  getPeakTimes(daysBack)         → Get peak player times
  searchEvents(searchTerm)        → Full-text search events
  getServerStats()               → Get server summary
