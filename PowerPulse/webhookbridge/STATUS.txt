🚀 DEINE TO-DO (5 MINUTEN)
═════════════════════════════════════════════════════════════════════════════

1. Discord Webhooks erstellen
   └─ Gehe zu deinem Discord Server
   └─ Rechtsklick auf Channel → Edit Channel → Webhooks
   └─ New Webhook → Namen eingeben
   └─ "Copy Webhook URL" klicken
   └─ URL speichern (sieht aus wie: https://discord.com/api/webhooks/...)

2. main.js konfigurieren
   └─ Öffne: D:\BB\bridgePlugins\webhookbridge\main.js
   └─ Zeile ~25: Webhook-URLs eintragen statt "YOUR_ID_HERE"
   └─ Speichern

3. Server neu starten
   └─ /reload oder Server neustarten

4. Testen
   └─ Im Chat: !webhook test
   └─ Prüfe Discord auf Test-Nachricht ✅

FERTIG! 🎉
