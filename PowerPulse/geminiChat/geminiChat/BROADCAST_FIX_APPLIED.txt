================================================================================
HOW IT WORKS
================================================================================

Broadcasting (world.sendMessage vs player.sendMessage):

OLD:
  player.sendMessage(message)  // Only player sees it

NEW:
  if (broadcastResponses) {
    world.sendMessage(message)  // EVERYONE sees it
  } else {
    player.sendMessage(message)  // Only player sees it
  }

Custom Prefix:

OLD:
  formatMultilineResponse(text)  // Always uses [Gemini]

NEW:
  const prefix = getConfig("responsePrefix", "BlockAI")
  formatMultilineResponse(text, prefix)  // Uses configured name

================================================================================
NEW CONFIG OPTIONS
================================================================================

1. responsePrefix (String)
   Default: "BlockAI"
   Examples: "BedrockBridgeAI", "ServerAI", "AIHelper"
   Where it appears: Response messages, errors, typing indicator

2. broadcastResponses (Boolean)
   Default: true
   If true: Everyone sees responses (world.sendMessage)
   If false: Only requester sees (player.sendMessage)

================================================================================
HOW TO CONFIGURE
================================================================================

Edit config.js:

Line 17: responsePrefix: "BlockAI"
  Change to:
  responsePrefix: "BedrockBridgeAI"
  or
  responsePrefix: "MyCustomAI"

Line 18: broadcastResponses: true
  Set to false if you want responses private

That's it! Restart server and changes apply.
