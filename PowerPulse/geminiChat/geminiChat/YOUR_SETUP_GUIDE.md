## 🚀 JETZT BRAUCHEN SIE NUR:

### Schritt 1: Plugin aktivieren
Starten Sie Ihren Minecraft Bedrock Server und geben Sie als Admin ein:

```
/plugin enable ./bridgePlugins/geminiChat/main
```

Sie sollten in der Konsole sehen:
```
[Gemini] Ready! Type @g <question> to chat
```

### Schritt 2: Sofort verwenden!

Im Spiel einfach tippen:
```
@g How do I craft a diamond pickaxe?
```

**DAS WAR'S! ✓**

Der AI wird antworten:
```
[Gemini] Here's how to craft a diamond pickaxe...
```

## ✨ Bonustipps

### Bessere Fragen stellen
```
❌ Bad:  @g How to farm?
✅ Good: @g What's the best farm design for automatic iron collection?
```

### Temperature für verschiedene Antworten
```
@g /gemini ui → Settings → Temperature

0.5  = Factual (für Fakten)
0.7  = Balanced (EMPFOHLEN - Standard)
1.0  = Creative (für Ideen)
1.5  = Very Creative (für Geschichten)
```

### Mehrere Spieler verwenden
```
Jeder Spieler kann gleichzeitig @g nutzen!
Jeder hat seine eigene Konversation.
Keine Konflikte!
```

## 🆘 Troubleshooting

### "Chat funktioniert nicht"
```
Lösung: /plugin reload
Warten Sie 10 Sekunden
Versuchen Sie erneut
```

### "Keine Antwort vom AI"
```
Lösung: /gemini status
→ Prüfe ob API Key aktiv ist
```

### "Zu viele Anfragen"
```
Lösung: Sie sind rate-limited
Warten Sie 1-2 Minuten
Dann funktioniert es wieder
```
